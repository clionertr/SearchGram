## Quick Start
1.clone to your vps
  install Redis and meilisearch
2.`sudo apt install python3 python3-venv`
  `python3 -m venv venv`
  `source venv/bin/activate`
  `python client.py`and`python bot.py`
3.use `screen` to leave keyboard
